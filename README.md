# 285 플래너

챌린저스 EXP 패스, 모멘텀 패스, 몬스터파크, 에테리온 아티팩트와 하드 메이린 보상 가치를 함께 계산하는 반응형 성장 전략 사이트입니다.

## 기본 시나리오

- 시작: Lv.280 87.390% · 2026-07-17
- 0주 · 9/16 마감만: 8/22 → 몬파 15,000MP
- 1주 당김: 8/19 → 몬파 51,000MP
- 추가 비용: 36,000MP = 14.4억 메소
- 하드 1회 가치: 4.92억 메소 · 회수율 34.2%

평일 7판 종료일은 고정하지 않습니다. 화면의 주차 버튼을 누르면 9월 16일 마감을 지키는 범위에서 직전 단계보다 하드 메이린을 한 주 더 확보하는 최소 몬파 경로를 다시 찾습니다. 메포샵 메카베리·블루베리는 평일 몬파보다 우선순위가 낮아 추천과 손익에서 제외하고 비교 옵션으로만 제공합니다.

계산 조건과 이벤트 수령 상태는 화면에서 모두 변경할 수 있습니다.

## 실행

```bash
pnpm install
pnpm dev
```

프로덕션 빌드는 `pnpm build`로 검증합니다.
